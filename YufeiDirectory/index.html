Research Interests
     My research interests are software analysis, software security, software engineering, programming languages, and energy efficient building computing.

Current Projects

  • JCut: JCut helps users create trustworthy computing environments by cutting off user-undesired features from Java software by using interprocedural static analysis based on system dependency graph(SDG).
  • JRed: JRed removes unused methods and classes from Java bytecode to reduce the size and improve the security of Java Program via combining interprocedural static data flow Analysis and dynamic analysis.
  • Babel: Babel obfuscates the software program by mixing two programming-paradigm-heterogeneous languages.
  • BIMserver Query Generator: BIMserver Query Generator automatically generates Java source code to help energy efficient building designer perform BIMserver query by parsing architecture engineering domain formatted documents.
